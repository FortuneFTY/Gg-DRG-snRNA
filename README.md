# Gg-DRG-snRNA

## Overview
This repository contains analysis code for chicken DRG snRNA-seq data, including clustering, marker identification, TRPA1-related analysis, differential expression, GO enrichment, etc.

## Data
Data were generated from DRG samples under control and HC conditions.

## Workflow
Main steps include quality control and filtering, normalization, PCA, clustering, UMAP/t-SNE visualization, doublet removal, sample integration, marker gene identification, TRPA1-positive cell analysis, differential expression analysis, GO enrichment, etc.

## Requirements
Main R packages include Seurat, DoubletFinder, dplyr, ggplot2, ggrepel, clusterProfiler, org.Gg.eg.db, etc.

## Notes
Some steps involve local file handling and may require adjustment according to the local environment.
